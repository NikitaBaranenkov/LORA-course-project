# Artifacts

Этот каталог предназначен для локальных артефактов обучения: checkpoints,
сохранённых tokenizer/model, архивов экспортов и других тяжёлых файлов,
необходимых для воспроизведения или продолжения конкретного запуска.

Текущий ноутбук DistilBERT сохраняет модельные артефакты в:

```text
artifacts/distilbert_full_ft/
├── checkpoints/
└── best_model/
```

Содержимое `artifacts/`, кроме этого README, исключено из Git. В репозитории
следует хранить только небольшие итоговые метрики в `reports/tables/` и графики
в `reports/figures/`.
