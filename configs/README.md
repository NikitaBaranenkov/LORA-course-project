# Experiment Configs

Каталог предназначен для YAML/JSON-конфигураций воспроизводимых экспериментов.
На текущем этапе параметры существующего запуска находятся непосредственно в
ноутбуке `notebooks/experiments/distilbert_full_finetuning.ipynb`; перенос
параметров в конфиги следует выполнять вместе с выделением общего training
pipeline в `src/`.
